A responsive side navigation with dropdown functionality

Welcome to the Test Page

This page demonstrates a fully functional side navigation menu with smooth animations and a responsive design. Click the hamburger menu in the top-left corner to open the navigation.

The navigation includes:

  • Smooth slide-in/slide-out animation
  • Dropdown menu for hobbies section
  • Hover effects on menu items
  • Responsive design that works on all screen sizes
  • Clean, modern styling

You can customize the colors, sizes, and animations by modifying the CSS variables to match your website's theme.

How to Use This Navigation

To use this navigation in your own project:

  1. Copy the HTML structure of the navigation
  2. Include the CSS styles in your stylesheet
  3. Add the JavaScript function to handle the toggle functionality
  4. Customize the menu items and styling to fit your needs
