App Development

In this class, I learned how to build cross-platform mobile applications for both Android and iOS. The course focused heavily on front-end development and creating intuitive, user-friendly interfaces.

Key Learnings:

Cross-Platform Mobile Development

I learned the entire process of developing mobile apps, from initial idea to final publication.

Human-Centered Design

This approach emphasized involving users from the start of the design process and keeping their needs central throughout development. It was about creating not just functional apps but ones that truly understood and met user expectations.

Development Tools

  • JavaScript on Node.js: I explored how Node.js extends JavaScript’s capabilities beyond browser scripting, making it a versatile language for various applications.
  • React & React Native: I used React to build user interfaces and React Native to develop cross-platform apps. React’s component-based architecture helped manage state and render UIs effectively, while React Native allowed me to translate JavaScript into native commands on mobile devices.
  • Expo: This tool made development smoother by providing pre-configured templates and a library of ready-to-use components. It also simplified testing and deployment with its development server and Expo Go app.
  • Firebase: I used Firebase to handle backend configuration and content management. It was great for quickly building MVPs and keeping costs low during the prototyping phase.

Overall, the course provided a solid foundation in mobile app development and introduced me to essential tools and practices in the field.
